# 🏦 Bank 24h — Sistema de Autoatendimento Bancário

> Projeto prático desenvolvido em HTML, CSS e JavaScript, simulando um terminal de autoatendimento bancário com múltiplos bancos, autenticação, operações financeiras validadas e geração de extrato.

---

## 📋 Sobre o Projeto

A **Abc technology** identificou falhas críticas em seus sistemas anteriores: usuários inseriam dados inválidos nos campos financeiros, causando erros no banco de dados. O sistema **Bank 24h** foi desenvolvido para resolver esse problema com uma camada de validação rigorosa usando **do-while** e **Expressões Regulares (REGEX)**.

---

## ✨ Funcionalidades

- ✅ Seleção de banco com troca dinâmica de cor de fundo
- ✅ Login com autenticação por usuário e senha por banco
- ✅ Ver saldo formatado em BRL
- ✅ Saque com validação REGEX + regras de saldo
- ✅ Depósito com validação REGEX
- ✅ Transferência com validação REGEX + regras de saldo
- ✅ Troca de senha
- ✅ Extrato com histórico de transações e data/hora
- ✅ Geração de PDF do extrato
